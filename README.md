#Game of Life

Game of life developed with React.js. Part of FCC projects.

[Demo page](http://vadimdez.github.io/react-game-of-life/)

### Run dev

Install dependencies

```
npm install
```

run server

```
npm run serve
```

Open [http://localhost:9000](http://localhost:9000) in browser.

### Uses

* React
* Redux
* SCSS
* Babel
* Webpack
